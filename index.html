CUSTOM WEB FONTS

HOW TO LOAD CUSTOM WEB FONTS

LOCALLY HOSTED

A few different ways!

  1. Custom Web Font > Download the woff files and .css
  2. If it's a TTF or OTF > then check license > if it's open-source and non-copyrighted, then use a webfont convertor to generate a web font package.
  3. Add your woff font files to a /fonts subfolder of your website project.
  4. Add a fontname.css file to your /css subfolder
  5. Ensure your font names are declared in your CSS and targeted in your styles

Right click > Inspect to see how to integrate into your project.
